What is the definition of server port? 
Answer: A port number is a way to identify a specific process to which,
        an Internet or,a other network message is to be forwarded,
        when it arrives at a server.

What is server port range?
Answer: Registered ports are in the range 1024 to 49151. 
        Dynamic ports are in the range 49152 to 65535. 

How many bits does number above represent?
Answer:  port number uses 16 bits and so,
         can therefore have a value from 0 to 65535 decimal.
